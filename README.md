# Package Structure


```
taskify
├─ pages                    # 페이지
├─ public
│  ├─ icons                 # 아이콘 리소스
│  └─ images                # 이미지 리소스
├─ src
│  ├─ apis                  # API 사용을 위한 세팅
│  ├─ components            # 주요 컴포넌트
│  ├─ constants             # 페이지 설정
│  ├─ contexts              # 인증, 모달 컨텍스트 프로바이더
│  ├─ hooks                 # 커스텀 훅
│  ├─ types                 # 주요 Json 타입
│  └─ util                  # 기타 유틸리티
├─ static
│  └─ fonts                 # 폰트
└─ styles                   # 스타일
``` 

# Installation
1. Clone the repository
```bash
git clone https://github.com/codeit-fe2-2/taskify.git
```
2. Install dependencies
```bash
npm install
```
3. Start the development server
```bash
npm start dev
```
4. Open the project in your browser
```bash
http://localhost:3000
```
